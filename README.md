# Smart Meter JSON API

RxJS based reader & REST API for Dutch Smart Meter/P1 port

# Install

- Checkout
- Create a .env file

```conf
HOST=0.0.0.0
PORT=2000
P1_HOST=smart-meter
P1_PORT=23
INTERVAL_MOVING_AVERAGE=60
```

- Run

```sh
npm install
npm start
```

- Open http://localhost:2000/api/gas or http://localhost:2000/api/electric
